Maximum number of USB ports

This is the maximum number of USB plugs BurnInTest will attempt to connect to. A max of 20 plugs can be used at the same time.

 

Run benchmark test during cycle

This will run a short benchmark test as part of each USB test cycle and the max read and write speeds will be logged. The benchmark test will only run run on USB2 plugs.
